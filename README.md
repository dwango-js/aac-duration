# aac-duration
aac duration getter

## Installation
```sh
$ npm install -g aac-duration
```

## Usage
### as command line tool

```sh
$ aac-duration hoge.aac
1.984sec
```

### as node module

```js
var duration = require("aac-duration");
console.log(duration("hoge.aac"));
```
